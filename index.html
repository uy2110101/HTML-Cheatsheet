HTML Cheatsheet For You Guys !
  1. HTML BoilerPlate 
    
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
  2. First Heading Tag 
    
<h1> this is main heading </h1>
  3. Subheadings 
    
<h2> this is second heading</h2>
<h3> this is third heading</h3>
  4. Image tag with source 
    
<img src="image address" alt="TextHere">
  5. Anchor Tag 
    
<a href="linkHere"></a>
  6. Bold and Italic Text Tags 
    
<b>This is bold</b>
<i>This is italic</i>
  7. Subscript and Superscript 
    
<sub>This is subscript</sub>
<sup>This is superscript</sup>
  8. Big and Small 
    
<big>This is big</big>
<small>This is small</small>
  9. To Add a video with url 
    
<video src="linkHere">Title of video</video>
  10. Audio or song tag 
    
<audio src="audioLink">AudioTitle</audio>
